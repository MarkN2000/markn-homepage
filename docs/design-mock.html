高機能VR日本語キーボード「MarkNキーボード」導入・使い方【Resonite】

ResoniteではVRキーボードを好きなものに変更できるって知っていましたか？　外部でなにか導入することなく、Resoniteの中だけでサクッと日本語キーボードに変更できますよ。

この記事では、Resoniteで使える高機能日本語キーボード「MarkNキーボード」の導入・使い方・機能をまとめています。デスクトップモードの場合は通常どおり手元の物理キーボードから直接入力できるので、この記事はVRモードでプレイする人向けの内容です。

導入方法

導入はとても簡単で、パブリックフォルダからキーボードを取り出して装備するだけです。以下の手順で進めてください。

  • ダッシュメニューからインベントリを開く
  • フォルダのリンクをワールド内にペーストする
  • キーボード本体を装備(Equip)する

紹介動画も用意しています。実際の入力の様子は動画のほうが分かりやすいと思います。

主な機能

MarkNキーボードには、日本語入力のほかにも変換候補の学習、定型文、電卓などの機能があります。設定はキーボード上の言語設定キーからいつでも変更できます。

ローマ字入力と変換

入力はローマ字方式で、スペースキーで変換候補を切り替えます。よく使う単語は自動で学習され、次回から上位に表示されるようになります。

変換エンジンはワールド内のロジックだけで動いているので、外部サービスとの通信は発生しません。

開発者向け: 出力の受け取り

キーボードの出力は動的変数経由で受け取れます。自作アイテムと連携する場合は次のように参照します。

// キーボードの確定文字列を受け取る
DynamicValueVariable<string> output;
output.VariableName = "MarkNKeyboard/CommittedText";
OnValueChanged(output, HandleText);
